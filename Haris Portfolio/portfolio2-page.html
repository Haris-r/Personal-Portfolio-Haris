Games Review MVC website

PHP framework (Codeigniter)

CodeIgniter is an Application Development Framework - a toolkit - for people who build web sites using PHP. Its goal is to enable you to develop projects much faster than you could if you were writing code from scratch, by providing a rich set of libraries for commonly needed tasks, as well as a simple interface and logical structure to access these libraries.

CodeIgniter lets you creatively focus on your project by minimizing the amount of code needed for a given task.

A framework used to create the front-end for the chat box was Vue-JS, the server was implemented using Socket IO which allowed me to create a real time server where you could interact with anyone within the local machine.

The goal to create this website was for users to be able to navigate through different featured games coming from a MySQL database, after choosing their favourite games they could look at the different information that came with the game, its general review directly from IGN and most importantly they could leave a comment which would update dynamically without needing to refresh the page
