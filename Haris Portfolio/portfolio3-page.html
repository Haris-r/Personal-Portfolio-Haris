Movie Database Presentation Tool

JAVA, JavaScript & JQuery

Developing, deploying, publishing and finding your own web service is the name of this project.

The goal of this project was to create an old movies presentation using JAXB and GSON libraries to convert the data, using CRUD functionalities to create, read, update and delete the data and manipulating it using AJAX requests so the user can manipulate the database easily using the implemented jQuery user interface.

I have uploaded the database to google cloud and tested it so whenever any user would want to instantiate the program, they can easily do so by a normal app engine server without having to worry about implementing their own database.
